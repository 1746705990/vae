# Vae / 那个夏天的播放器

> “有何不可，为何不可。”

这是一个简简单单的在线音乐播放器，写给自己，也写给那些回不去的夏天。
没有复杂的数据库，没有繁琐的登录，只有纯粹的音乐和那抹记忆中的渐变色。

## 🎵 关于这个项目

最初只是想做一个安静听歌的地方。
它有着毛玻璃般的质感，像透过雾气看过去的时光。
支持切换不同的歌单（比如那些年的翻唱，和最初的原创），支持单曲循环——哪怕只是为了听清某一句歌词。

### 特性

* **毛玻璃界面**：梦幻般的紫蓝渐变，干净通透。
* **双曲库切换**：默认支持“早期翻唱”与“早期原创”两个篇章，一键穿越。
* **随心播放**：支持顺序播放、随机播放、单曲循环。
* **自动收录**：不需要手动输入歌名，只要把 MP3 放进文件夹，它就会自动把它们找出来。

## 📂 食用方法

不需要复杂的安装，只要你有一个支持 PHP 的小空间。

### 1. 准备环境

把所有文件上传到你的服务器。

### 2. 找回丢失的旋律

因为一些原因，原本的音乐文件不在压缩包里。你需要手动创建两个文件夹，把你的珍藏放进去：

* **新建文件夹 `music1`** -> 放入 **早期翻唱** 的 MP3 文件。
* **新建文件夹 `music2`** -> 放入 **早期原创** 的 MP3 文件。

*(文件名会自动成为歌名，请把它们重命名得整整齐齐)*

### 3. 开始聆听

打开浏览器，访问 `index.php`。
带上耳机，闭上眼。
